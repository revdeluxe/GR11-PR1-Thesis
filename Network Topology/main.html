Bus Topology

A bus network is a network topology in which nodes are directly connected to a common half-duplex link called a bus.

Star Topology

A star network connects to a central device. This central device may be a hub or a switch.

Mesh Topology

A mesh network are multiples nodes that are interconnected to each other. former a network which is everyone is a host.

Tree Topology

A tree network is a special type of structure where many connected elements are arranged like the branches of a tree.

Hybrid Topology

A Hybrid topology is an integration of two or more different topologies to form a resultant topology.
